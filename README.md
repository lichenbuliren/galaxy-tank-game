# 🌌 Galaxy Tank

Galaxy × Tank Wars 混合休闲游戏

## 玩法
- 移动端全屏：手指左右滑动控制战机
- 自动射击，无需按键
- 击落 **Tank 类型敌机** 可获得 BUFF 道具

## BUFF 道具
- 🛡️ **防护盾** — 短暂无敌
- 🚀 **弹速+** — 射速翻倍
- 💣 **散弹** — 一次发射3发
- ⏱️ **减速** — 敌人移动变慢
- ❤️ **生命+1**

## 技术
- 纯 HTML5 Canvas，无依赖
- Web Audio API 合成 MIDI 音效 + BGM
- 移动端 touch 手势适配
